# Commands

```bash
ElmerGrid 14 2 hinge.msh -autoclean
```

Can be found in manual section 1.3:
- Elmergrid [Input file format] [Output file format] [Input file name]

## Input format
- Lots of possible meshfile formats generated by different meshers / CAD formats

## Output format
Different formats of elmer software, can be
- .grid (ElmerGrid format)
- .mesh (ElmerSolver format): input is divided into 4 different files
	- mesh.boundary
	- mesh.elements
	- mesh.header
	- mesh.nodes
- .ep: ElmerPost file format: post-processing

There are other undocumented formats
